# Changelog

All notable changes to `taapi-go` will be documented in this file.

## [1.0.0] - 2026-02-01

### Added
- Initial release
- Full support for taapi.io API v1
- GET (Direct) request support with fluent interface
- POST (Bulk) request support for multiple indicators
- POST (Manual) request support for custom candle data
- Type-safe constants for Exchanges, Indicators, and Intervals
- Custom error handling (Error, RateLimitError)
- Response types (IndicatorResponse, BulkResponse)
- Comprehensive test coverage
- Support for Go 1.21+
- Zero external dependencies for core functionality
- Complete GoDoc documentation
